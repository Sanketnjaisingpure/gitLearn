Git quick notes

- Initialize repo: git init
- Clone a repo: git clone <url>
- Check status: git status
- Stage changes: git add <file> | git add .
- Commit changes: git commit -m "message"
- View history: git log --oneline --graph --decorate --all

Branching
- List branches: git branch
- Create branch: git branch <name>
- Switch branch: git switch <name>   (or)   git checkout <name>
- Create + switch: git switch -c <name>
- Merge into current: git merge <branch>

Remote
- Add remote: git remote add origin <url>
- Rename default branch locally: git branch -M main
- Push first time: git push -u origin main
- Pull latest: git pull

Undo/Recover
- Unstage file: git restore --staged <file>
- Discard local changes: git restore <file>
- New commit from previous: git commit --amend -m "new message"
- See where HEAD was: git reflog
- Reset to commit (soft): git reset --soft <commit>
- Reset to commit (hard, destructive): git reset --hard <commit>

Stash
- Save work: git stash push -m "note"
- List stashes: git stash list
- Apply latest (keep): git stash apply
- Pop latest (remove): git stash pop

Helpful configs
- Set name: git config --global user.name "Your Name"
- Set email: git config --global user.email "you@example.com"
- Default editor: git config --global core.editor "code --wait"

Tips
- Use small, frequent commits with clear messages.
- Always pull before pushing.
- Create feature branches instead of committing on main.
